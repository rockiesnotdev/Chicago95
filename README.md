Basically Chicago95 with some patches to make it match Windows XP Classic Theme.

## Requirements:
- Xfce 4.20
- gtk2-engines-pixbuf (Recommended for GTK2 applications)

-----

## Installation

Just copy everything inside Theme/ dir to /usr/share/themes/ or .themes
Just copy everything inside Icons/ dir to /usr/share/icons/ or .icons
